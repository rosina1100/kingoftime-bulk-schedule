# King of Time 一括スケジュール登録

King of Time（勤怠管理システム）のスケジュール登録画面に「一括スケジュール登録」機能を追加するChrome拡張です。

## こんな人向け

- SES企業で、毎月のスケジュール登録を手作業でやっている管理者
- 社員ごとに客先が異なり、勤務パターンが複数ある
- King of Timeの自動スケジュール機能では対応しきれない

## できること

コピー元の特定の1日（例: 5月1日）のスケジュールパターンを、対象月（例: 7月）の全平日に一括適用します。

- **複数パターン対応**: 社員ごとに異なるスケジュールパターンがあっても、1回の保存で全パターンをまとめて登録
- **スキップ/上書きモード**: 既にスケジュールが入っている日をスキップするか上書きするか選択可能
- **プレビュー機能**: 実行前に適用件数・スキップ件数を確認
- **安全設計**: チェックボックスの選択まで自動化し、保存はユーザーが目視確認してから手動で実行

## インストール

1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」からこのフォルダを選択

## 使い方

1. King of Timeのスケジュール登録画面を開く
2. 「表示」ボタンの右隣に表示される「一括スケジュール登録」ボタンをクリック
3. コピー元の日付（年/月/日）を指定
4. 「プレビュー」で適用内容を確認
5. 「一括チェック」でチェックボックスが自動選択される
6. 画面上でチェック状態を確認し、KOTの「保存」ボタンを押す

## 対象環境

- Chrome（Manifest V3）
- King of Time: `https://s2.ta.kingoftime.jp/admin/*`

## ファイル構成

```
├── manifest.json          # Chrome拡張マニフェスト（V3）
├── content.js             # メイン（UI注入・オーケストレーション）
├── content.css            # UIスタイル
├── lib/
│   ├── kot-parser.js      # KOTページDOM解析
│   ├── schedule-logic.js  # ビジネスロジック
│   └── kot-api.js         # API通信・チェックボックス操作
├── background.js          # Service Worker
└── icons/                 # 拡張アイコン
```
